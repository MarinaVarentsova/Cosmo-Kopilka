# Космо-копилка 🚀

Семейная копилка с космической темой — трекер баланса, целей и сложного процента. Работает полностью в браузере, данные хранятся в localStorage.

## Возможности

- Баланс с анимированной планетой-копилкой
- Цели с прогресс-баром и прогнозом накопления
- Сложный процент — таблица и 10-летний график роста
- Курсы ЦБ РФ в реальном времени (USD, EUR, CNY)
- Дневник транзакций (пополнение / расход / процент)
- Сценарии прогноза на 1, 6 и 12 месяцев
- Видео-демо из 5 сцен (`/video.html`)
- Полностью адаптивный дизайн (мобильный, планшет, десктоп)

## Стек

- **Backend:** Node.js + Express (только для раздачи статики и проксирования курсов ЦБ)
- **Frontend:** чистый HTML + CSS + JS в одном файле (`public/index.html`)
- **Хранилище:** localStorage (без базы данных и авторизации)

## Запуск локально

```bash
npm install
npm start
```

Приложение откроется на [http://localhost:5000](http://localhost:5000)

## Деплой на Railway / Render / Fly.io

1. Форкни репозиторий
2. Создай новый проект на платформе
3. Укажи команду запуска: `npm start`
4. Укажи переменную окружения `PORT` (если нужно; по умолчанию 5000)

## Деплой на GitHub Pages (статика без API курсов)

Скопируй `public/index.html` и `public/video.html` в корень ветки `gh-pages`. Курсы ЦБ будут использовать встроенные резервные значения.
