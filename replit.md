# Космо-копилка

A space-themed family savings tracker app in Russian. Tracks balance, savings goals, transactions, interest projections, and a 10-year growth chart.

## Architecture

- **server.js** — simple Express static file server
- **public/index.html** — complete single-page app (HTML + CSS + JS, all self-contained)

## Running

```bash
npm start
```

Runs on port 3000.

## User preferences

- Language: Russian UI
- Self-contained single HTML file for the frontend
